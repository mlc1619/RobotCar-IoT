🚗 RobotCar-IoT

Sistema completo para controlar un robot móvil con ESP8266 + L9110S desde un dashboard web con Node.js, WebSocket en tiempo real y MySQL.

Ver código en GitHub Documentación

Autor: Misael López Cancino · Mayo 2026

¿Por qué esta página no controla el robot?

GitHub Pages sirve solo archivos estáticos. El control real requiere:

  • Servidor Node.js + WebSocket corriendo en tu PC
  • Base de datos MySQL local
  • Estar en la misma red WiFi que el ESP8266

👉 Para usar el dashboard real, clona el repo y sigue la guía local.

✨ Características

Control en tiempo real

Comandos vía WebSocket con latencia < 100 ms.

CRUD de dispositivos

Administración completa de robots registrados en MySQL.

Parámetros ajustables

Velocidad, sensibilidad y tiempos configurables al vuelo.

Demos automatizadas

Secuencias pregrabadas reproducibles desde el panel.

Monitoreo en vivo

Estado del robot, conexión y telemetría en tiempo real.

Historial

Últimos 10 movimientos y demos ejecutadas persistidas en BD.

🚀 Quick Start (local)

# 1. Clonar el repositorio
git clone https://github.com/mlc1619/RobotCar-IoT.git
cd RobotCar-IoT/server

# 2. Configurar la base de datos
mysql -u root -p < sql/schema.sql

# 3. Configurar variables de entorno
cp .env.example .env
# editar .env con tus credenciales MySQL

# 4. Instalar dependencias y arrancar
npm install
node server.js

# 5. Abrir el dashboard
# http://localhost:3000
